아키텍처 핵심 설계

성능

1회 Fetch

staleTime: Infinity 설정으로 앱 생애 주기 내 단 1회만 API 호출. 보상 수령 시에만 수동 invalidate로 재조회.
안정성

Graceful Degradation

API 실패 / 데이터 없음 시 isRewardEligible()이 true 반환. 서버 오류가 유저 경험 방해 없음.
통합

단일 API

5가지 보상 상태를 GET 1회로 일괄 조회. 기존 개별 API 호출은 하위 호환 유지. 추후 보상 키 추가 시 타입만 확장.
확장성

Barrel Export

free-reward/index.ts 배럴 익스포트로 타입, API, 훅을 단일 진입점에서 임포트 가능.
