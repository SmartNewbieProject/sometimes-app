### 1.2 개발 공수 산정 방식

**MM (Man-Month)**: 1개월 = 20.6일 = 164.8시간

**복잡도별 개발 시간 기준**:
- **간단**: 코드 100줄당 0.5~1시간
- **보통**: 코드 100줄당 1~2시간
- **복잡**: 코드 100줄당 2~4시간

**추가 고려 사항**:
- 기획/설계: 개발 시간의 20%
- QA/테스트: 개발 시간의 30%
- 버그 수정: 개발 시간의 15%
- **총 실제 소요 시간 = 개발 시간 × 1.65**

### 1.3 외주 에이전시 적용 배율

**출처**: 업계 평균 마진율 조사
