## 1\. 하지 않을 이유 제거 (Objection Handling)

**신뢰/안전 우려**

* "이상한 사람 만나면 어떡해?" → 대학생 이중 인증 (학생증 + 학교 이메일)
* "가짜 프로필 아냐?" → 인스타그램 연동, 실물 검증
* "지인한테 들키면?" → 같은 학교/학과 차단 기능

**가격 부담**

* "돈 아까워" → 무응답 시 100% 환불 보장
* "다른 앱보다 비싸면?" → 월 4,900원 (커피 한 잔 값)

**시간/노력 우려**

* "가입 귀찮아" → 인스타 연동으로 프로필 자동 생성
* "매칭 안 되면?" → 대전 11개 대학 활성 유저풀

**심리적 장벽**

* "소개팅앱 쓴다고 티나면?" → 대학생 전용이라 동질감
* "연애할 준비 안 됐는데" → 친구부터 시작 가능

## 2\. 왜 우리여야 하는지 (Value Proposition)

**기능적 이유**

* 같은 지역 대학생만 → 실제로 만날 수 있음
* 가치관/취향 기반 매칭 → 외모만 보는 앱이 아님
* 매칭 설명 제공 → 왜 이 사람인지 알 수 있음

**감성적 이유**

* "대학생 시절에만 가능한 설렘"
* "우리 동네, 우리 학교" 편안함
* 친구가 소개해주는 느낌의 진정성

**타이밍 이유 (크리스마스 캠페인)**

* 종강 직후 → 시간 여유 생김
* 크리스마스 → 솔로 심리 자극
* 새 학기 전 → 새로운 시작 동기

## 3\. 행동 허들 낮추기 (Friction Reduction)

**가입 단계**

* 카카오/애플 원클릭 로그인
* 인스타에서 사진 자동 가져오기
* 프로필 작성 가이드/예시 제공

**첫 사용 단계**

* 무료 좋아요 3회 제공 (체험)
* 웰컴 보상 (구슬)
* AI 썸메이트가 첫 대화 도움

**결제 단계**

* 소액 시작 (커피 한 잔 프레이밍)
* 무응답 시 환불 → 리스크 제거
* "한 달만 써보기" 심리적 부담 완화
