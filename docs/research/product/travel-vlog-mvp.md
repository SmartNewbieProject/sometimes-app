좋습니다.

“해외여행 브이로그 MVP”를 **① 반응 나오는 구조 템플릿** + **② 자동화까지 고려한 편집 구조 설계** 두 축으로 정리합니다.

목표는 **CTR + 초반 유지율 + 평균 시청시간** 검증입니다.

---

# **① 반응 나오는 여행 브이로그 구조 템플릿 (10–12분 기준)**

## **🎬 전체 구조 개요**

| **구간** | **시간** | **목적** | **구성 원칙** |
| -- | -- | -- | -- |
| Hook | 0:00–0:20 | 클릭 보상 | 가장 강한 장면 3컷 |
| Context | 0:20–1:00 | 왜 이 여행인가 | 감정 + 기대 |
| Core 1 | 1:00–4:00 | 첫 임팩트 | 먹방/풍경/사건 |
| Core 2 | 4:00–8:00 | 감정선 | 갈등·피곤·웃김 |
| Emotional Close | 8:00–10:00 | 기억에 남기기 | 회상/정리 |

---

## **🔥 0:00–0:20 Hook 설계 (가장 중요)**

구성 공식:

> 강한 장면 → 예상 밖 반응 → 한 줄 멘트

예:

* “이거 실화야?” (자막)
* 가장 예쁜 풍경 1초
* 음식 한입 리액션
* 갑자기 비 맞는 장면

⚠️ 설명 금지

⚠️ 자기소개 금지

⚠️ “안녕하세요\~” 금지

---

## **🎧 BGM 설계 구조**

| **구간** | **BGM 톤** |
| -- | -- |
| Hook | 빠른 템포 |
| 이동/설명 | 가벼운 lo-fi |
| 감정선 | 따뜻한 피아노 |
| 엔딩 | 여운 |

---

## **✍ 자막 전략**

자막은 3종류만:

1. 감정 자막

   → “진짜 기대 안 했는데…”
2. 상황 자막

   → “새벽 5시 공항”
3. 리액션 자막

   → “이건 미쳤다”

전체 문장 자막 ❌

핵심 단어만 강조.

---

**② 자동화를 고려한 편집 구조 설계**

지금 MVP부터 **자동화 가능한 구조**로 만들어야 합니다.

## **🧠 Claude/Codex 활용 흐름 (MVP 수준)**

### **Step 1 — 영상 스크립트 생성**

원본 대사 → LLM이 정리:

```
이 여행의 감정선을 5단계로 정리해줘
```

### **Step 2 — Hook 후보 5개 생성**

```
초반 20초에 쓸 강한 장면 조합 5개 만들어줘
```

### **Step 3 — 자막 요약**

```
이 영상에서 자막으로 넣을 핵심 문장만 추려줘
```

### **Step 4 — 제목 A/B 10개**

```
유튜브 CTR 높을 제목 10개
```

# **🎯 지도 / 그래픽 MVP 기준**

### **지도는 이렇게만:**

* 시작할 때 3초
* “한국 → 일본”
* 선 애니메이션

그 이상 ❌

그래픽은:

* 가격 정보
* 위치 정보
* 환율

정도만.

---

# **🧪 유튜브 테스트 설계**

## **A/B 테스트 방법**

### **1차 테스트 (같은 영상)**

* 썸네일 2개
* 제목 2개

### **2차 테스트**

* Hook 다른 버전 업로드 (비공개 실험)
