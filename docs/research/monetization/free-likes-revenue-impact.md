좋은 데이터야. 이 정도면 **“감으로 무료화”가 아니라, 숫자로 의사결정**할 수 있어.

아래는 **현재 구슬 소비 구조를 기준으로**,
👉 **‘프로필 좋아요 무료화’ 시 매출 타격이 얼마나 되는지**
👉 그리고 **실제로는 왜 타격이 제한적일 가능성이 높은지**를 단계적으로 분석한 결과야.

---

## 1️⃣ 현재 구슬 매출 구조 요약 (팩트 정리)

### 🔹 전체 구슬 소비

* **총 소비량**: **191,922개**
* 이 중 기능별 비중:

| 기능 | 소비 비중 |
| -- | -- |
| 재매칭 | **69.8%** |
| 프로필 좋아요 | **14.1%** |
| 구 재매칭 | **11.4%** |
| 프로필 열기 | 2.9% |
| 채팅 시작 | **1.5%** |
| 기타 | 0.3% |

👉 **핵심 포인트**

* 좋아요는 **소비 비중 14.1%**
* 채팅 시작은 \*\*1.5%\*\*로 매우 작음
* **재매칭(총 81.2%)이 사실상 매출의 엔진**

---

## 2️⃣ “좋아요 무료화” 시 **직접적인 매출 감소**

### 🔻 사라지는 소비

* **프로필 좋아요 소비량**
  * **27,096개 (14.1%)**

👉 **이론상 최대 매출 감소 상한선 = 14.1%**

하지만 이건 **최악의 가정**임
(= 좋아요를 유료로 쓰던 모든 사용자가 완전히 결제를 중단한다고 가정)

---

## 3️⃣ 현실적으로는 왜 “14% 풀 감소”가 안 나올까?

### 이유 1️⃣ 좋아요 소비자는 이미 “저가 소비층”

| 지표 | 수치 |
| -- | -- |
| 좋아요 사용자 수 | 815명 |
| 회당 비용 | 6.5개 |
| 사용 횟수 | 4,148회 |

👉 이 유저들은

* **소액·저빈도**
* **실험적 사용**
* 아직 Aha Moment 이전

즉,

> “이 사람들이 좋아요를 안 사면, 대신 아무것도 안 사게 된다” ❌
> “이 사람들이 좋아요 무료 → 다음 단계로 이동한다” ⭕️

---

### 이유 2️⃣ 현재는 “돈을 써도 성과가 안 나오는 구조”

현재 구조:

* 좋아요 결제 → **대화 확률 1%**
* 즉, **99%는 ‘돈 쓰고 아무 일도 없음’**

이 상태에서 좋아요 매출은 사실상:

> 🔥 **신뢰를 소모하면서 벌어들이는 매출**

이건 장기적으로 유지 불가능한 수익이야.

---

## 4️⃣ 무료화 후 **대체 소비가 발생하는 지점**

### 🔄 전환 구조 변화 (중요)

#### Before (현재)

```
유저
 └─ 좋아요(유료) ❌ 대부분 안 누름
      └─ 매칭 ❌
           └─ 채팅 ❌
```

#### After (좋아요 무료화)

```
유저
 └─ 좋아요(무료) ⬆️⬆️
      └─ 매칭 ⬆️⬆️
           ├─ 채팅(유료) ⬆️
           ├─ 재매칭 ⬆️⬆️
           └─ 프로필 열기 ⬆️
```

---

## 5️⃣ 숫자로 보는 “회수 가능한 매출”

### 📌 시나리오 가정 (보수적)

* 좋아요 무료화 → 좋아요 사용 **3배 증가**
* 매칭률 상승 → 재매칭 사용 **10%만 증가**
* 채팅 시작 소비 **3배 증가**

---

### 🔼 증가분 계산

#### ① 재매칭

* 현재: **133,970개**
* +10% → **+13,397개**

#### ② 채팅 시작

* 현재: **2,866개**
* ×3 → **+5,732개 (+2,866)**

#### ③ 프로필 열기 (보수적으로 +20%)

* 5,587 → **+1,117개**

---

### 🔻 감소분

* 좋아요 무료화 손실: **-27,096개**

---

### ✅ 순변화 계산

```
+13,397 (재매칭)
+2,866 (채팅)
+1,117 (프로필 열기)
—————————
= +17,380개

-27,096 (좋아요)
—————————
= 🔻 -9,716개
```

👉 **총 소비 대비 약 -5.1%**

---

## 6️⃣ 그런데 이 -5%도 “실제론 과대평가”

왜냐면:

### 🔹 재매칭은 이미 **핵심 소비**

* 감정 몰입이 생기면 **지출 탄력성 매우 높음**
* “대화 경험 후 재매칭”은 현재 거의 없음 → 성장 여지 큼

### 🔹 장기 효과는 계산에 안 들어감

* 리텐션 상승
* 결제 유저 수 증가
* 구슬 패키지 구매 전환
* “돈 써도 안 되는 앱” → “돈 쓰면 되는 앱” 인식 전환

👉 이건 **단기 매출 -5% vs 구조 개선**의 문제야.

---

## 7️⃣ 결론 (의사결정용 요약)

### ❌ 하지 말아야 할 판단

> “좋아요 매출 14% 날아가니까 위험하다”

### ✅ 데이터 기반 결론

* **실질적 단기 타격: 약 -3 \~ -7%**
* **중기(1\~2개월) 내 회복 가능**
* **구조적으로는 필수적인 수술**

### 🎯 핵심 한 문장

> **지금의 좋아요 매출은 ‘성과 없는 과금’에서 나오는 매출이고,**
**무료화는 그 돈을 ‘성과 있는 소비’로 이동시키는 구조 전환이다.**

* A: 기존
* B: 좋아요 무료 + 채팅 유료 유지

2️⃣ 지표 추적

* 좋아요 사용률
* 매칭률
* 채팅 오픈율
* 재매칭 소비 변화
* PUR 변화

3️⃣ 2주만 돌려도 결론 나옴

원하면
👉 **A/B 실험 설계안**,
👉 **투자자용 한 장 요약**,
👉 **“단기 매출 방어용 가격 미세 조정안”**
바로 만들어줄게.

지금 단계에선 **결정해도 되는 수준의 데이터**야.
