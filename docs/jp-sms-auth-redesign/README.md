## 1. 개선 배경

### 1.1 현황 분석

일본 시장 진출을 위해 SMS 인증 기능을 구현했으나, 현재 UI/UX가 다음과 같은 문제점을 가지고 있습니다:

| 문제점 | 영향 |
|--------|------|
| 사무적/딱딱한 문구 | 브랜드 아이덴티티와 불일치, 이탈률 증가 우려 |
| Progress 바 부재 | 진행 상황 불명확, 사용자 불안감 유발 |
| 한국 온보딩과 UI 불일치 | 브랜드 일관성 저하 |
| 신뢰/설렘 요소 부재 | 회원가입 첫 단계에서 기대감 미형성 |

### 1.2 개선 목표

1. **브랜드 일관성 확보**: 한국 온보딩과 동일한 UI 패턴 적용
2. **사용자 경험 개선**: 친근하고 설레는 문구로 전환
3. **신뢰 요소 강화**: 보안 관련 안내 명확화
4. **진행 상황 시각화**: Progress 바로 완료 가능성 인지

## 3. AS-IS vs TO-BE 비교

### 3.1 전화번호 입력 화면

| 항목 | AS-IS | TO-BE |
|------|-------|-------|
| **제목** | 電話番号を入力してください | 素敵な出会いまで、あと少し！ |
| **부제** | 認証コードをSMSで送信します | 電話番号を教えてね。安全な認証のためだけに使うよ 🔐 |
| **버튼** | 認証コードを送信 | 認証コードを受け取る |
| **Progress** | 없음 | 50% 표시 |
| **배경** | 단색 | 그라디언트 (한국 동일) |
| **+81 스타일** | 기본 회색 박스 | 브랜드 컬러 강조 |

### 3.2 인증코드 입력 화면

| 항목 | AS-IS | TO-BE |
|------|-------|-------|
| **제목** | 認証コードを入力 | もうすぐだよ！ ✨ |
| **부제** | {{phone}}に送信されたコードを入力してください | {{phone}}に届いた6桁のコードを入れてね |
| **버튼** | 確認 | 出会いをはじめる |
| **타이머** | 残り時間: 2:30 | あと 2:30 (뱃지 스타일) |
| **재발송** | コードを再送信 | 届いてない？もう一度送る |
| **Progress** | 없음 | 100% 표시 |

  - Step 2 (코드 확인): 100%

## 5. 브랜딩 문구 변경 상세

### 5.1 톤앤매너 가이드라인

| 기존 | 개선 | 이유 |
|------|------|------|
| ~してください (명령) | ~してね (친근) | 친구같은 느낌 |
| ~します (사무적) | ~するよ (대화체) | 카카오톡 느낌 |
| 기능 중심 설명 | 감정 중심 표현 | 설렘 유발 |

### 5.2 문구별 의도

| 문구 | 의도 |
|------|------|
| 素敵な出会いまで、あと少し！ | 설렘 + 완료 근접 암시 |
| 電話番号を教えてね | 친근함 (친구에게 물어보듯) |
| 安全な認証のためだけに使うよ 🔐 | 신뢰 + 투명성 |
| もうすぐだよ！ | 격려 + 기대감 |
| 出会いをはじめる | 단순 "확인"이 아닌 의미 부여 |
| 届いてない？もう一度送る | 친근한 질문형으로 부담 완화 |

## 6. 인터랙션 정의

### 6.1 전화번호 입력

1. **입력 시작**: 포커스 시 +81 박스 강조
2. **자동 포맷팅**: 숫자 입력 시 `090-1234-5678` 형태로 자동 변환
3. **유효성 검사**: 070/080/090으로 시작하는 11자리 확인
4. **버튼 활성화**: 유효한 번호 입력 시 버튼 활성화

### 6.2 인증코드 입력

1. **타이머**: 3분 카운트다운, 뱃지 스타일로 표시
2. **타이머 만료**: 재발송 링크로 전환
3. **자동 포커스**: 화면 진입 시 코드 입력란 자동 포커스
4. **완료 처리**: 6자리 입력 완료 시 버튼 활성화

## 7. 에러 처리

| 에러 케이스 | 기존 메시지 | 개선 메시지 |
|-------------|------------|-------------|
| 잘못된 번호 | 正しい電話番号を入力してください | 正しい電話番号を入力してね |
| SMS 발송 실패 | SMS送信に失敗しました | SMSの送信に失敗しちゃった。もう一度試してみて |
| 잘못된 코드 | 認証コードが正しくありません | コードが違うみたい。もう一度確認してね |
| 코드 만료 | 認証コードの有効期限が切れました | コードの有効期限が切れちゃった。再送信してね |
