# 호칭 컴포넌트 화면 제목
1 칭호 카드 TitleAwardSection —
2 스탯 바 AnimatedStatRow 나의 연애 성향
3 레이더 차트 RadarChart 성향 전체 지도

5 스토리 카드 StoryCard 답변으로 보는 나
6 성장 힌트 GrowthJourneySection 앞으로의 성장 힌트
7 페르소나 카드 PersonaNarrativeSection 나의 페르소나

1 칭호 카드 · TitleAwardSection

보라 그라디언트 배경에 큰 칭호 텍스트와 부제목을 표시. 우상단·좌하단에 반투명 장식 원형 포함. 항상 화면 최상단에 위치.

props: ceremonyText title subTitle LinearGradient #7A4AE2→#5B35C0 borderRadius 20px 장식 원형 2개

2 스탯 바 · AnimatedStatRow × 5

5개 성향(외향성·개방성·성실성·친화성·정서적 안정)을 세로 목록으로 표시. 각 행은 큰 숫자(36px 900weight) + 아이콘 + 이름 + 그라디언트 바. 진입 시 순차 슬라이드-인 애니메이션(index × 120ms 딜레이).

props: category, score, change, label, index 애니메이션: withDelay + withTiming 바 그라디언트 #7A4AE2→#C4A8F5 score ≥70 검정, ≥50 주황, 미만 빨강 흰 컨테이너 borderRadius 16px

3 레이더 차트 · RadarChart

오각형 방사형 SVG 차트. 5개 성향을 꼭짓점으로 배치하며 격자선 5단계(20/40/60/80/100). 데이터 폴리곤은 보라 색상으로 채우고 꼭짓점에 원형 점 표시. 차트 하단에 RadarLegend로 수치 바 표시.

props: radarData (RadarDataItem[]) 채움 rgba(122,74,226,0.2) 격자 #E5E5E5 흰 카드 래퍼 borderRadius 20px 주간 리포트: 이전 주 점선(#A0A0A0) 추가

4 패턴 인사이트 · IntegratedInsightsSection

두 요소로 구성. 위: 태그 칩 행(keyPatterns 배열). 아래: 왼쪽 보라(3px) 테두리 인용구 블록(content 텍스트). 칩은 옅은 보라 배경에 보라 텍스트.

props: keyPatterns[], content 태그 배경 #F8F5FF, 테두리 #D4C2F5 인용 블록 borderLeft 3px #7A4AE2

5 스토리 카드 · StoryCard × N

N개의 카드 반복. 각 카드: 제목 → 인용 칩(보라 둥근 태그, userQuote.highlight) → 스토리 텍스트 블록(옅은 보라 배경) → 감정 여정 화살표(chip → chip). 흰 카드, 가벼운 그림자.

props: title, userQuote, userStory, emotionalJourney[], whatThisTellsUs 인용 칩 배경 #EDE9F8, 테두리 #C4ADFF 본문 블록 #F8F5FF borderRadius 10px 여정 칩 borderRadius 12px

6 성장 힌트 · GrowthJourneySection

nextSteps 배열의 각 항목을 카드로 표시. 왼쪽에 번호 배지(보라 원, 흰 숫자), 오른쪽에 텍스트. 흰 카드 + 가벼운 그림자. 하단에 선택적 suggestion 텍스트(회색 이탤릭).

props: nextSteps[], suggestion? 배지 28×28 borderRadius 14 #7A4AE2 카드 borderRadius 14px

7 페르소나 카드 · PersonaNarrativeSection

단일 흰 카드. 상단 description 텍스트(회색, 줄높이 24px) 이후 characteristics 배열을 체크리스트로 표시. 체크 아이콘(✓)은 보라색, 텍스트는 검정.

props: description?, characteristics[] 체크 ✓ 색상 #7A4AE2 카드 borderRadius 16px 항목 간격 gap 8px

개발 참고 사항

📁
공유 컴포넌트 위치
7개 섹션 컴포넌트 모두 src/features/moment/ui/shared/narrative-report-components.tsx 한 파일에 정의됨. 온보딩 결과와 주간 리포트 양쪽에서 import.
🔀
Narrative 모드 분기
주간 리포트(weekly-report-page.tsx)는 report.reportType === 'narrative' 조건으로 분기. Legacy 모드는 별도 UI 사용.
🎨
배경 그라디언트
페이지 전체 배경: LinearGradient #FFFFFF → #FAFAFE → #F2EEFF → #E4DAFF (absolute fill). 카드들은 불투명 흰 배경으로 구분.
✨
스탯 바 애니메이션
AnimatedStatRow는 react-native-reanimated의 withDelay + withTiming 사용. index × 120ms 순차 딜레이, 상하 슬라이드 + 페이드인.
🌐
i18n 섹션 제목 키
스탯: result.statsTitle ("나의 연애 성향")
레이더: result.radarChartTitle ("성향 전체 지도")
패턴: result.patternsTitle ("내가 가진 연애 패턴")
스토리: result.storyTitle ("답변으로 보는 나")
성장: result.growthTitle ("앞으로의 성장 힌트")
페르소나: result.personaTitle ("나의 페르소나")
📊
성향 5가지 순서 (고정)
DIMENSION_ORDER 상수: extraversion → openness → conscientiousness → agreeableness → neuroticism. 순서 변경 시 이 상수 수정.
