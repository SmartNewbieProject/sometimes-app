Retry & Debounce Logic

중복 모달 방지, 재시도 메커니즘, 자동 닫힘의 상세 동작

⏲ Debounce: 3초 내 중복 요청 방지
문제 상황: 화면 로드 시 여러 API가 동시에 호출되고, 서버 장애 시 모두 502를 반환. debounce 없이는 N개의 모달이 연달아 표시될 수 있다.
1 3초 윈도우 시각화
0.0s
GET /users → 502 → emit → showModal()
0.1s
GET /matches → 502 → emit → BLOCKED (3s window)
0.3s
GET /chat → 502 → emit → BLOCKED (3s window)
0.5s
GET /profile → 502 → emit → BLOCKED (3s window)
3.1s
(윈도우 만료) 새 502 수신 시 → showModal() 허용
모달 표시됨 차단됨 (3s 이내) 윈도우 만료 후 허용
2 구현 코드 패턴
const DEBOUNCE_MS = 3000;
const lastShownRef = useRef<number>(0);

// eventBus 리스너 내부
const now = Date.now();
if (now - lastShownRef.current < DEBOUNCE_MS) {
  return; // 3초 이내 중복 → 무시
}
lastShownRef.current = now;
↻ Retry Logic: 재시도 메커니즘
1 "다시 시도" 클릭 시 동작 (502만 해당)
1. 실패한 요청 config 저장
eventBus emit 시 전달받은 AxiosRequestConfig를 ref에 저장
2. "다시 시도" 클릭
hideModal() 호출 후, 저장된 config로 axiosClient(config) 재실행
3-A. 재시도 성공 (2xx)
모달 닫힘 유지. 앱 정상 플로우 복귀. 저장된 config 초기화
3-B. 재시도 실패 (502/503 재발)
모달 재표시 + 메시지 변경: "계속 오류가 발생하고 있어요"

Persistent Error 상태: 재시도도 실패했을 때. 모달을 다시 표시하되, 본문 메시지를 "계속 오류가 발생하고 있어요.\n잠시 후 다시 시도해주세요."로 변경하여 사용자에게 지속적 장애임을 인지시킨다.
3 Auto-dismiss: 성공 시 모달 자동 닫힘
동작: "다시 시도" 버튼 클릭 → hideModal() 즉시 호출 → 요청 재시도 시작.
성공하면 모달은 이미 닫혀 있으므로 추가 동작 불필요.
실패하면 eventBus.emit('server:maintenance')가 다시 발생하여 모달 재표시.
// "다시 시도" 버튼 핸들러
const handleRetry = async () => {
  hideModal();
  try {
    await axiosClient(lastFailedConfigRef.current);
    // 성공 → 모달 이미 닫힘, 정상 복귀
    lastFailedConfigRef.current = null;
  } catch {
    // 실패 → interceptor가 다시 emit → 모달 재표시
    // debounce window를 리셋하여 즉시 표시 허용
    lastShownRef.current = 0;
  }
};
⚠ Edge Cases
A 502 → 503 전환
502로 모달을 표시했는데 재시도 시 503이 반환되면, 모달을 503 variant(서버 점검)로 교체한다. eventBus payload의 type 필드로 구분.
B 모달 표시 중 다른 API 성공
502 모달 표시 중 다른 API 호출이 성공하더라도 모달은 유지된다. 모달은 사용자의 명시적 액션(버튼 클릭)으로만 닫힌다.
C 오프라인 상태와의 구분
네트워크 연결 끊김(no response)과 502/503(서버 응답 있음)은 다른 상태. error.response가 존재해야 502/503 처리로 진입. error.response가 없으면 기존 네트워크 에러 핸들링 유지.

import { useEffect, useRef } from 'react';
import { useTranslation } from 'react-i18next';
import { eventBus } from '../libs/event-bus';
import { useModal } from './use-modal';
import axiosClient from '../libs/axios';
import type { AxiosRequestConfig } from 'axios';

const DEBOUNCE_MS = 3000;

type MaintenancePayload = {
  type: 'temporary' | 'maintenance';
  config: AxiosRequestConfig;
};

export function useServerMaintenanceModal() {
  const { t } = useTranslation();
  const { showModal, hideModal } = useModal();
  const lastShownRef = useRef(0);
  const lastConfigRef = useRef<AxiosRequestConfig | null>(null);

  useEffect(() => {
    const unsub = eventBus.on(
      'server:maintenance',
      (payload: MaintenancePayload) => {
        const now = Date.now();
        if (now - lastShownRef.current < DEBOUNCE_MS) return;
        lastShownRef.current = now;
        lastConfigRef.current = payload.config;

        if (payload.type === 'maintenance') {
          showMaintenanceModal();
        } else {
          showTemporaryErrorModal();
        }
      }
    );
    return unsub;
  }, [showModal, hideModal, t]);
}
