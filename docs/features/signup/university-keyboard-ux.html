기본 상태 검색 팁이 표시됩니다.
이전/다음 버튼이 하단에 고정되어 있습니다.
tip 문구가 제거되어 화면이 깔끔합니다.
검색 중 상태 키보드가 올라오면:
· 검색 팁 숨김 (max-height)
· 버튼이 translateY(50) + opacity 0
· 리스트 공간이 최대화
선택 완료 상태 대학교 탭 시:
· Keyboard.dismiss() 즉시 호출
· 키보드 내려가며 버튼 복귀
· 다음 버튼 활성화
키보드 dismiss 트리거 흐름

↕️

버튼 복귀
다음 활성화
keyboardDismissMode="on-drag" — 리스트 스크롤 시 자동 dismiss
keyboardShouldPersistTaps="handled" — 탭 이벤트 정상 전달
Keyboard.dismiss() — 대학 선택 onClick에서 즉시 호출
1
tip 문구 제거
배경과 겹쳐 읽기 어렵던 "학교 인증을 통해..." 문구 삭제. HelpIcon import, tipContainer/tip 스타일 함께 제거.
2
버튼 페이드아웃
키보드 올라올 때 translateY(50) + opacity:0 애니메이션(200ms). 기존 bottomButtonsAnimatedStyle 활용.
3
스크롤로 키보드 내리기
keyboardDismissMode="on-drag" 추가. 리스트를 드래그하면 키보드가 자연스럽게 내려가고 버튼이 복귀.
4
선택 시 즉시 dismiss
대학교 카드 클릭 시 Keyboard.dismiss() 즉시 호출. 선택→키보드 내림→버튼 활성화 흐름이 자연스러움.
