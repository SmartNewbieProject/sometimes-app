Comparison
현재 vs 개선 후 비교
항목 현재 (Walla 외부 링크) 개선 후 (인앱 편지 모달)
유저 이탈 ✗ 외부 브라우저로 이탈 ✓ 앱 내 바텀시트 유지
Slack 알림 ✗ 없음 ✓ POST /feedback → Slack Block Kit
카테고리 분류 ✗ 없음 ✓ 일반/버그/기능/기타 칩 선택
완료율 ✗ 낮음 (이탈 후 미완료) ✓ 높음 (모달 내 원스텝)
UX 일관성 ✗ Walla 타 브랜드 UI ✓ 모먼트 편지와 동일 패턴
확장성 ✗ 없음 ✓ 구슬 보상 · 어드민 연동 가능
API Spec
POST /feedback 연동 스펙
Public 엔드포인트. 인증 불필요. 성공 시 204 No Content.
// Request POST /feedback Content-Type: application/json { "message": "앱이 전반적으로 좋아졌어요!", // Required. Max 2000자 "category": "general" // Optional. general|bug|feature|other } // Response (성공) 204 No Content // Body 없음 // Slack 수신 메시지 (Block Kit) 📣 유저 피드백이 도착했어요! 카테고리: 💬 일반 내용: "앱이 전반적으로 좋아졌어요!" // 에러 케이스 400 — message 누락 또는 2000자 초과 400 — category가 허용값 외 문자열 500 — Slack API 호출 실패
