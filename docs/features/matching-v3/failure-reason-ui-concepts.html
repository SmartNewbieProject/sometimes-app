추천 방향
B안 (Centered Bubble)을 메인으로 추천합니다.
- 데이팅 앱의 따뜻하고 부드러운 톤과 잘 맞고, failureCode별 컬러 분기가 자연스러움
- FILTERED_OUT일 때 CTA 버튼이 카드 내부에 자연스럽게 통합
- failureCode가 없으면 기존 UI 그대로 유지 (하위 호환성 완벽)

빠른 MVP가 필요하면 C안 (Inline Text) — 서버 failureReason만 표시하고 끝.
사유 강조가 중요하면 D안 (Dark Toast) — 시선 유도력이 가장 강함.
