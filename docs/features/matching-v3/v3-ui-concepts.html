5가지 시안 비교

시안 구현 난이도 기존 UI 변경 리매칭 발견성 정보 밀도 학습 비용 핵심 장단점
A. Info Strip
BEST + 직감적, 항상 보임
- 카드 영역 살짝 줄어듦
B. Carousel
중간 + 카드 크기 100% 유지
- secondary 놓칠 수 있음
C. PIP
낮음 + 유튜브처럼 직감적, 시선 고정
- PIP가 사진 일부 가림
D. Segment
낮음 + 발견성 최고, count 뱃지
- 탭 전환 필요 (추가 동작)
E. Bottom Sheet
낮음 + 기존 BottomSheet 재활용
- 하단 공간 차지, 카드 줄어듦
추천 조합
1순위 — A + D 하이브리드: Info Strip의 항상-보이는 정보 + Segmented Tab의 높은 발견성 결합. strip 대신 segmented control을 사용하되, 비활성 탭에 미니 타이머 표시.
2순위 — E (Bottom Sheet): 기존 앱 BottomSheetPicker 패턴 완전 재활용. 개발 비용 최소.
3순위 — C (PIP): 시각적 임팩트 최고. 유튜브 패턴으로 학습 비용 낮음. 단, 프로필 사진 가림 이슈.
