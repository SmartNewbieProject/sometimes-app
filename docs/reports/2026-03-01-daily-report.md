## Executive Summary

> 매칭 실패 시 사용자에게 **구체적인 실패 사유와 해결 방법**을 안내하는 기능이 프론트엔드·백엔드 양쪽 모두 구현 완료되었습니다. 기존에는 단순히 "매칭된 사람이 없습니다"로만 표시하던 것을, 3가지 원인 유형(회원 수 부족, 필터 설정, 시스템 오류)에 따라 8가지 세분화된 메시지로 분류하여 사용자 이탈을 방지합니다.

---

## 주요 성과 — 매칭 실패 사유 안내 기능

### 1. 기능 개요

매칭 결과가 없을 때 **"왜 실패했는지"**와 **"무엇을 하면 되는지"**를 사용자에게 즉시 안내하여, 서비스에 대한 신뢰와 재시도 동기를 높이는 기능입니다.

### 2. 실패 원인 분류 체계 (8가지 케이스)

| 유형 | 케이스 | 사용자 메시지 예시 | 예상 빈도 |
|------|--------|-------------------|----------|
| **회원 부족** | 지역 내 부족 | "{지역명}에서 현재 매칭 가능한 상대가 없어요. 조금만 기다려주세요!" | 높음 |
| **회원 부족** | 재매칭 과다 (50회 이상) | "이미 많은 분들과 매칭을 진행하셨어요. 새로운 회원이 가입하면 다시 시도해보세요." | 중간 |
| **회원 부족** | 일반 부족 | "현재 매칭 가능한 상대가 없어요." | 낮음 |
| **필터 설정** | 같은 학교 회피 | "같은 학교 회피 설정으로 인해 매칭 가능한 상대가 없어요. 필터를 해제하면 더 많은 분들을 만날 수 있어요." | 중간 |
| **필터 설정** | 같은 학과 회피 | "같은 학과 회피 설정으로 인해 매칭 가능한 상대가 없어요." | 낮음 |
| **필터 설정** | 연락처 차단 | "연락처 차단 설정으로 인해 매칭 가능한 상대가 없어요." | 낮음 |
| **필터 설정** | 복합 필터 | "설정하신 필터 조건으로 인해 매칭 가능한 상대가 없어요." | 낮음 |
| **시스템 오류** | 일시적 오류 | "일시적인 문제가 발생했어요. 잠시 후 다시 시도해주세요." | 매우 낮음 |

### 3. 사용자 화면 구성

매칭 실패 화면에 다음 요소가 표시됩니다:

- **실패 사유 토스트**: 화면 상단에 보라색 배경의 안내 카드로 실패 원인을 표시
  - 원인별 아이콘 (🔍 회원 부족 / ⚙️ 필터 설정 / 💑 재매칭 과다)
  - 필터 설정이 원인인 경우 → **"필터 설정 →"** 바로가기 버튼 제공
- **3가지 행동 유도 카드**:
  - **새로운 매칭 시도** → 재매칭 버튼
  - **매칭 조건 확장** → 이상형 범위 수정 화면 이동
  - **프로필 개선** → 사진/자기소개 수정 화면 이동

### 4. 백엔드 분석 시스템

서버에서 매칭 파이프라인 각 단계의 데이터를 기반으로 자동 원인 분석을 수행합니다:

- **후보 검색 단계**: 검색 결과 0건 → 회원 수 부족으로 분류, 사용자 지역 및 기존 매칭 이력(50회 기준)에 따라 세분화
- **필터링 단계**: 후보가 있었으나 필터로 전원 제거됨 → 어떤 필터가 가장 많은 후보를 제거했는지 분석하여 주원인 특정
- **로그 기록**: 모든 실패 건에 대해 원인 카테고리, 코드, 필터 적용 전후 후보 수, 제거율, 필터별 제거 수, 지역 이성 풀 크기 등을 DB에 기록

### 5. 비즈니스 임팩트

| 항목 | 기대 효과 |
|------|----------|
| **사용자 이탈 방지** | 실패 원인 이해 → 재시도 의지 향상 |
| **필터 최적화 유도** | 과도한 필터 사용자에게 완화 제안 → 매칭 성공률 개선 |
| **운영 인사이트** | 실패 로그 데이터 축적 → 지역별·원인별 매칭 품질 모니터링 가능 |
| **고객 문의 감소** | "왜 매칭이 안 되나요?" 유형 문의 사전 차단 |

### 6. 다국어 지원

한국어, 일본어, 영어 3개 언어로 모든 실패 메시지가 번역되어 글로벌 서비스에도 대응 완료되었습니다.
