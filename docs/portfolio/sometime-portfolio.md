> **작성일**: 2026-01-27
> **프로젝트명**: 썸타임 - 대학생 지역 기반 소개팅 플랫폼

---

## 1. 서비스 개요

### 1.1 서비스 소개
**썸타임**은 20대 대학생을 위한 지역 기반 소개팅 앱입니다. 학교 인증 시스템을 통해 신뢰도 높은 매칭을 제공하며, 매주 목/금 21시에 이상형 매칭이 진행됩니다.

### 1.2 핵심 기능
- **이상형 매칭**: 매주 정해진 시간에 AI 기반 매칭 진행
- **학교 인증**: 대학생 신분 인증을 통한 안전한 매칭 환경
- **커뮤니티**: 자유 게시판, 인기글, 후기 등 다양한 소통 공간
- **모먼트**: 데일리 룰렛, 썸메이트 등 다양한 이벤트 기능
- **지인 차단**: 연락처 기반 지인 매칭 제외 기능

### 1.3 주요 지표
- **누적 신청자**: 79,440명 이상
- **타겟 사용자**: 20대 대학생
- **매칭 주기**: 매주 목/금 21시

## 2. 기술 스택

### 2.1 Mobile App (React Native)

| 기술 | 버전 | 용도 |
|------|------|------|
| **Expo** | SDK 54 | 크로스 플랫폼 개발 환경 |
| **React Native** | 0.81 | 모바일 UI 프레임워크 |
| **TypeScript** | 5.x | 타입 안전성 |
| **Zustand** | 5.x | 경량 상태 관리 |
| **TanStack Query** | 5.x | 서버 상태 관리 |

### 2.2 Web App (Next.js)

| 기술 | 버전 | 용도 |
|------|------|------|
| **Next.js** | 15 | App Router 기반 웹 프레임워크 |
| **React** | 19 | UI 라이브러리 |
| **TypeScript** | 5.x | 타입 안전성 |
| **Tailwind CSS** | 4.x | 유틸리티 기반 스타일링 |

### 2.3 아키텍처
- **FSD (Feature-Sliced Design)**: 기능 중심의 모듈화된 아키텍처
- **모바일/웹 동시 지원**: Expo + Next.js 멀티 플랫폼

## 4. 기술적 특징

### 4.1 모바일 앱
- **Expo SDK 54**: 최신 Expo 기능 활용
- **FSD 아키텍처**: 기능별 모듈화로 유지보수성 향상
- **Biome**: 코드 포맷팅 및 린팅 자동화

### 4.2 웹 앱
- **App Router**: Next.js 15 최신 라우팅 시스템
- **Server Components**: 서버 사이드 렌더링 최적화
- **반응형 디자인**: 모바일 웹 최적화

### 4.3 보안
- **학교 인증 시스템**: 대학생 신분 검증
- **지인 차단**: 연락처 기반 프라이버시 보호
- **안심 매칭**: 지인과 마주치지 않는 매칭
