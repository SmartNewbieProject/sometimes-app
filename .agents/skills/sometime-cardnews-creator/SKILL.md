# 썸타임 카드뉴스 제작 스킬

썸타임 브랜드 가이드라인을 준수하여 카드뉴스 콘텐츠를 제작합니다.

---

## 필수 워크플로우 (2단계)

### STEP 1: 주제 흐름 컨펌 (필수)

카드뉴스 작성 전 **반드시** 슬라이드별 주제 흐름을 먼저 제안하고 사용자 승인을 받습니다.

**제안 형식**:
```
📋 카드뉴스 주제 흐름 (컨펌 요청)

유형: [신규 기능 / 운영 업데이트 / 이벤트 / 교육]
타깃: [박서준 / 김지은 / 전체]
핵심 메시지: [한 문장]

---
| 슬라이드 | 주제 | 핵심 내용 (1줄) |
|---------|------|----------------|
| 1 | [주제] | [핵심 내용] |
| 2 | [주제] | [핵심 내용] |
| 3 | [주제] | [핵심 내용] |
| 4 | [주제] | [핵심 내용] |
---

이 흐름으로 진행할까요? 수정이 필요하면 말씀해주세요.
```

### STEP 2: 상세 콘텐츠 작성

사용자 승인 후 각 슬라이드의 title, content, 이미지 프롬프트를 작성합니다.

---

## 브랜드 보이스

### 톤앤매너
- **친근한 존댓말**: 친구가 진심으로 조언하듯, 선배가 후배를 챙기듯
- **3S 원칙**:
  - Simple: 중학생도 이해할 수 있게 쉽게
  - Sincere: 친구처럼 진심으로
  - Supportive: 함께 성장하며 응원

### 브랜드 가치
| 가치 | 의미 | 표현 방식 |
|------|------|----------|
| 진정성 | 솔직한 배경 설명, 실패 인정과 개선 약속 | "솔직히 말씀드릴게요" |
| 신뢰 | 검증 프로세스 투명 공개, 24시간 모니터링 | 데이터와 수치 제시 |
| 설렘 | 대학 시절 특별한 순간, 청춘 감성 | 기대감 조성 |
| 성장 | 과정 중심, 격려와 응원 | 작은 발전 인정 |

## 타깃 페르소나

### 박서준 (효율 지향)
- 공대 4학년, 데이터와 수치 중심
- **선호 키워드**: 효율, 시간 절약, 성공률, 확률, 검증, 데이터, 단계별
- **예시**: "평균 대화 시작 시간 3분 단축, 매칭 성공률 40% 증가"

### 김지은 (경험 지향)
- 인문대 2학년, 이야기와 경험 중심
- **선호 키워드**: 안전한, 편안한, 재미있는, 후기, 경험, 설레는, 특별한
- **예시**: "어색한 첫 대화, 이제 걱정 마세요. 친구들 사이에서 화제인 기능이에요!"

## 콘텐츠 유형

### 1. 신규 기능 공지 (4장)

| 슬라이드 | 목적 | 내용 |
|---------|------|------|
| 1. 티저 | 문제 인식 | 후킹 질문 + 문제 공감 + 해결책 힌트 |
| 2. 솔루션 | 기능 소개 | 기능명 + 핵심 가치 + 베네핏 |
| 3. 사용법 | 가이드 | 3단계 간결 가이드 + 팁 |
| 4. 마무리 | 응원/CTA | 따뜻한 응원 또는 행동 유도 |

### 2. 운영 업데이트 (3-4장)

| 슬라이드 | 목적 | 내용 |
|---------|------|------|
| 1. 변경 요약 | 무엇이/언제/누구에게 |
| 2. 변경 이유 | 진솔한 배경 + 유저 피드백 인용 |
| 3. 구체 내용 | Before → After 비교 |
| 4. 유저 액션 | 대응 방법 + FAQ (선택) |

### 3. 이벤트 프로모션 (3-5장)

| 슬라이드 | 목적 | 내용 |
|---------|------|------|
| 1. 티저 | 캐치프레이즈 + 기간 + 핵심 혜택 |
| 2. 혜택 | 혜택 리스트 + 조건 + 제한 사항 |
| 3. 참여법 | 3단계 이하 가이드 + 데드라인 |
| 4. 사회적 증거 | 참여자 수 + 후기 (선택) |
| 5. 재촉 CTA | 종료일 재강조 |

### 4. 교육 가이드 (4-5장)

| 슬라이드 | 목적 | 내용 |
|---------|------|------|
| 1. 문제 제기 | 공감 시나리오 + 통계 |
| 2-4. 솔루션 | Do & Don't + 예시 + 팁 |
| 5. 효과 격려 | 성과 수치 + 성공 사례 |

## 작성 규칙

### 문장 스타일
- 한 문장 = 한 아이디어 (15-25자, 최대 40자)
- 3문장마다 짧은 문장으로 리듬 조절

**권장 패턴**:
```
질문-답변: 대화가 시작되지 않으셨나요? 이제 환불받으실 수 있어요.
문제-해결: 매칭은 됐는데 답이 없다면, 구슬을 돌려드려요.
Before-After: 예전: 구슬 소진 / 지금: 환불 가능!
```

### 권장 어미
- ~예요 / ~이에요 (친근)
- ~해요 / ~할게요 (행동)
- ~하세요 / ~해보세요 (권유)
- ~죠? / ~지 않나요? (공감 유도)

### 금지 표현

**딱딱한 공문서체**:
| 금지 | 대체 |
|------|------|
| 고객님 | 여러분 |
| 이용자 | 썸타임 유저 |
| 본 서비스 | 썸타임 / 우리 |
| ~에 의거하여 | ~때문에 |
| ~하시기 바랍니다 | ~해주세요 |

**과장된 마케팅**:
| 금지 | 대체 |
|------|------|
| 대박! 혁명적! | 정말 유용해요 |
| 절대, 반드시 | 꼭 / 한번 |
| 100% 보장 | 더 높은 확률로 |

**부정적 프레이밍**:
| 금지 | 대체 |
|------|------|
| 실패, 불가능 | 아직 어려워요 |
| 안 됩니다 | ~가 더 좋아요 |
| 문제, 오류 | 개선할 점 |

**속어/신조어**: 킹받네, 레전드, 실화냐, ㅇㅈ, ㄹㅇ

### 숫자와 기호
- 10 이하: 한글 (세 번, 다섯 가지)
- 11 이상: 아라비아 숫자 (15명, 100구슬)
- 강조: 아라비아 숫자 (단 3일!)
- **사용**: • (불릿) → (인과) ✓/✗ (체크)
- **최소화**: !!! ??? ~~~
- **금지**: ㅋㅋ ㅎㅎ ♥♡★☆

### 이모지 사용
- title에 최대 1개, content에 최대 1개
- **권장**: 💡(팁) 🎁(혜택) 📢(공지) ✨(설렘) 💬(대화) 📍(포인트)
- **금지**: 💩 😡 😭 (부정) 🍆 🍑 (성적) 🤑 💰 (상업)

## 비주얼 가이드라인

### 이미지 사이즈
- **비율**: 4:5 (앱 내 카드뉴스 표준)
- **권장 사이즈**: 1080 x 1350px
- **앱 내 최대 높이**: 475px

### 색상
- **Primary**: #7A4AE2 (브랜드 퍼플)
- **Secondary**: #4ECDC4 (터콰이즈 - 신뢰)
- **Accent**: #FFE66D (옐로우 - 기쁨)
- **배경 권장**: 라벤더, 크림, 웜 화이트 그라데이션
- **배경 비권장**: 코럴 레드 (#FF6B6B)

### 이미지 스타일
- minimal flat illustration
- clean vector art
- romantic, warm mood
- **필수**: 이미지 내 텍스트 없음 (텍스트는 피그마에서 추가)

## 이미지 프롬프트 (나노/바나나용)

### JSON 형식

각 슬라이드에 아래 형식의 이미지 생성 프롬프트를 포함합니다:

```json
{
  "prompt": "이미지 설명 (영문)",
  "negative_prompt": "text, words, letters, typography, numbers, watermark, signature",
  "aspect_ratio": "4:5",
  "style": "minimal flat illustration, clean vector art, romantic mood"
}
```

### 프롬프트 작성 규칙
1. **영문으로 작성**
2. **텍스트 요소 배제**: negative_prompt에 text, words, letters 필수 포함
3. **배경색 명시**: soft lavender, cream, warm white gradient 등
4. **스타일 통일**: minimal flat illustration, clean vector art 유지
5. **감정 키워드**: romantic, hopeful, warm, gentle 등 포함

### 슬라이드별 이미지 컨셉 가이드

| 슬라이드 | 이미지 컨셉 | 추천 요소 |
|---------|------------|----------|
| 1. 티저 | 문제 상황 시각화 | 망설이는 손, 물음표, 고민하는 모습 |
| 2. 솔루션 | 기능의 핵심 가치 | 편지, 하트, 반짝임, 마법 효과 |
| 3. 사용법 | 단계별 플로우 | 아이콘 3개, 점선 연결, 화살표 |
| 4. 마무리 | 긍정적 결과/응원 | 연결된 하트, 종이비행기, 별 |

## 품질 체크리스트

### 톤 체크
- [ ] 친구가 말하는 것처럼 들리는가?
- [ ] 존댓말이지만 딱딱하지 않은가?
- [ ] 영업사원처럼 들리지 않는가?

### 명확성 체크
- [ ] 중학생도 이해할 수 있는가?
- [ ] 핵심 메시지가 3초 내 파악되는가?
- [ ] 다음 행동이 명확한가?

### 감정 체크
- [ ] 읽고 나서 기분이 좋아지는가?
- [ ] 설렘이나 기대감이 느껴지는가?

### 최종 질문
> "이 카드뉴스를 본 유저가 '아, 썸타임은 진짜 우리 편이구나'라고 느낄까?"
> - Yes → 발행
> - 잘 모르겠다 → 다시 쓰기
> - No → 처음부터 재작성

## 시작 프로토콜

카드뉴스 제작 요청 시 아래 순서를 따릅니다:

### 1. 정보 수집
- 어떤 유형의 카드뉴스인가요? (신규 기능 / 운영 업데이트 / 이벤트 / 교육)
- 주요 타깃은 누구인가요? (박서준 / 김지은 / 전체)
- 핵심 메시지를 한 문장으로 말씀해주시겠어요?
- 몇 장의 슬라이드를 원하시나요? (3-5장 권장)

### 2. 주제 흐름 제안 (STEP 1)
- 슬라이드별 주제와 핵심 내용을 표로 정리
- **사용자 컨펌 필수**

### 3. 상세 콘텐츠 작성 (STEP 2)
- 컨펌 후 각 슬라이드의 title, content, 이미지 프롬프트 작성
- 에디터 친화적 형식으로 출력

---

## 푸시알림 가이드

카드뉴스 발행 시 함께 발송하는 푸시알림을 작성합니다.

### 글자수 제한

| 항목 | 권장 | 최대 |
|------|------|------|
| **제목** | 15-25자 | 30자 |
| **본문** | 25-40자 | 50자 |

### 작성 규칙

1. **제목**: 호기심 유발 또는 핵심 가치 전달
2. **본문**: 구체적 내용 + 행동 유도
3. **이모지**: 본문 끝에 1개만 (선택)
4. **톤**: 카드뉴스와 동일 (친근한 존댓말)

### 유형별 템플릿

**신규 기능**:
| 스타일 | 제목 | 본문 |
|--------|------|------|
| 호기심 유발 | [핵심 가치]할 수 있다면? | [기능명]이 새로 생겼어요. [행동 유도] ✨ |
| 기능 직접 소개 | ✨ [기능명] 출시! | [핵심 기능 설명 1줄] |
| 문제 공감 | [문제 상황]했다면 | 이제 [해결책]할 수 있어요 💬 |

**운영 업데이트**:
| 스타일 | 제목 | 본문 |
|--------|------|------|
| 변경 안내 | [기능명]이 바뀌었어요 | [변경 내용 요약]. 확인해보세요! |
| 개선 강조 | 더 좋아진 [기능명] | [개선 포인트] 업데이트했어요 |

**이벤트**:
| 스타일 | 제목 | 본문 |
|--------|------|------|
| 혜택 강조 | 🎁 [혜택] 드려요 | [기간] 동안 [조건]하면 [혜택]! |
| 긴급성 | [D-N] [이벤트명] 마감 임박 | 아직 참여 안 하셨다면 서두르세요! |

**교육 가이드**:
| 스타일 | 제목 | 본문 |
|--------|------|------|
| 팁 제공 | 💡 [주제] 꿀팁 | [핵심 팁 1줄]. 자세히 알아보세요 |
| 문제 해결 | [문제]로 고민이라면 | [해결책]을 알려드릴게요 |

### 출력 형식

카드뉴스 출력 마지막에 푸시알림 옵션을 함께 제공합니다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📲 푸시알림 (택 1)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 옵션 A (호기심 유발) ⭐ 추천
제목: 좋아요에 마음을 담을 수 있다면?
본문: 편지 좋아요가 새로 생겼어요. 진심을 전해보세요 ✨

▶ 옵션 B (기능 직접 소개)
제목: ✨ 편지 좋아요 기능 출시!
본문: 좋아요와 함께 짧은 편지를 보낼 수 있어요

▶ 옵션 C (문제 공감)
제목: 좋아요만으론 부족했다면
본문: 이제 짧은 편지와 함께 마음을 전해보세요 💬
```

### 푸시알림 체크리스트

- [ ] 제목이 30자 이내인가?
- [ ] 본문이 50자 이내인가?
- [ ] 클릭하고 싶은 호기심이 생기는가?
- [ ] 카드뉴스 내용과 일관성이 있는가?
- [ ] 과장되거나 스팸처럼 느껴지지 않는가?
